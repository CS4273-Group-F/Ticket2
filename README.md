# React

## Using create-react-app

1. ```npx create-react-app fuel-station-locator```
2. ```cd fuel-station-locator```
3. In src, delete
    - App.test.js
    - logo.svg
    - reportWebVitals.js
    - setupTests.js
4. Edit
    - index.html
    - App.js
    - index.js
    - App.css
5. ```npm start```
