# React

## Using create-react-app

1. ```npx create-react-app demo-app_cra```
2. ```cd demo-app_cra```
3. In src, delete
    - App.test.js
    - logo.svg
    - reportWebVitals.js
    - setupTests.js
4. Edit
    - index.html
    - App.js
    - index.js
    - App.css
5. ```npm start```


## Using Vite
1. ```npm create vite@latest demo-app_vite```
2. name the project, select a framework and a variant
3. ```cd demo-app_vite```
4. ```npm install```
5. Edit
- index.html
- index.css
- App.jsx
- App.css
6. ```npm run dev```
